Promises in javascript

  • The Promise object represents the eventual completion (or failure) of an asynchronous operation.

  • A Promise is in one of these states:

    • pending: initial state, neither fulfilled nor rejected.

    • fulfilled: meaning that the operation was completed successfully.

    • rejected: meaning that the operation failed.

  • The pending promise can either be fulfilled with a value or rejected with a reason (error)

  • If promise is resolved you can handle it by .then() and if promise rejects. You can handle it by .catch(). Both .then(callback function) and .catch(callback function) takes callback functions. In which, data or error comes.

  • Example:

  • In above example, we have created a promise of registeration flow. In which, we are calling a register function which will register a user, after gettig a response of promise resolved. We have called email send function then login function and at last we call getUserData() function.

    But in getUserData() function we are rejecting the promise. So that we can know, how to handle it.

  • Like .then() and .catch(). We have other methods like Promise.all(), Promise.any(), Promise.finally(), Promise.race().

  • Promise.all() will handle all the promises. Which is passed by us in an array as an parameter in it. As shown in the below example.

    Promise.all() return us the array of resolved promises. Basically, it returns us a value Promise.all([register(),sendEmail(),getUserData()]).then((values)=>{ console.log(values); }).catch((e)=>{ console.log(e); })

    Example:

  • Promise.any() this method is used to return the first promise that fulfills.

    So as soon as a promise is fulfilled, it will not wait for other promises to complete.

    Example:

  • Promise.finally() will always run. It doesn't matter wheather the promise is fullfilled or reject. The finally will always run.

    Example

  • In the above example, We are adding two digits so after adding the numbers .then() will run and then immediately .finally() will also run.

  • Promise.race() The Promise.race() method returns a Promise that is resolved or rejected, as soon as one of the promises in an iterable, such as an array, fulfills or rejects, with the value or reason from that Promise.
